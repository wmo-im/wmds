This requirements class contains a set of rules applicable to the XML encoding of ObservingFacility.<br/><br/>An ObservingFacility could be a fixed station, moving equipment or a remote sensing platform. The definition of ObservingFacility describes the facility and it's surrounds. It does not describe the equipment at the facility (systems, sensors etc.). These are described using the Equipment class.<br/><br/>
